# vim
My vim development environment Settings

# 初衷:
	　1, 持续化改进个人工作环境;
	　2, 方便个人多开发环境的切换;
	　3, 方便个人多机器上的环境搭建与同步;

# 分支:
	empty: 系统配置
	basic:　基本配置
	master:　默认分支
	plugin: 插件集成
	probes: 探索尝试

	以上各分支之间并行存在， 不允许merge， 除非理解为什么要这样做．

# 使用
	执行以下:
	ln -s <YOU_PATH>/vim ~/.vim
	ln -s <YOU_PATH>/vimrc ~/.vimrc
